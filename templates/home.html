New York Events API Documentation

Introduction

This API provides information about various events happening in and around New York City. It allows users to create, view, and manage event details.

Base URL

The base URL is the root URL for all of the API's endpoints. All requests should start with this base URL:

https://api.newyorkevents.com

API Endpoints

POST /form

Use this endpoint to add a new event. The request should contain event details such as name, place, URL, image URL, and description.

GET /get-all-events

List all available events. This endpoint returns an array of all events with their details.

Example Request: https://api.newyorkevents.com/get-event/get-all-events

GET /get-event/<event_name>

Retrieve details of a specific event by providing the event name. This endpoint returns all relevant information about the event.

Example Request: https://api.newyorkevents.com/get-event/Fireworks

GET /get-event/<event_id>

Retrieve details of a specific event by providing the event id. This endpoint returns all relevant information about the event.

Example Request: https://api.newyorkevents.com/get-event/2

How to Use

Send HTTP requests to the above endpoints with the required parameters to interact with the API. Ensure to handle responses in your client application.
